## Agent Coordination Framework

### Tier 1: Core Writing Agents
**When to activate:**
- **Literary Architect**: Any narrative content, structure design
- **Truth Weaver**: Research needed, case studies, verification
- **Voice Alchemist**: Prose enhancement, AI pattern removal
- **Impact Engineer**: Frameworks, exercises, transformation design

**Typical workflow:**
1. Literary Architect → Structure
2. Truth Weaver → Research and evidence
3. Voice Alchemist → Prose polish
4. Impact Engineer → Actionability enhancement

### Tier 2: Specialized Content Agents
**When to activate:**
- **Blog Master**: Blog posts, SEO content
- **Social Media Strategist**: Threads, LinkedIn, viral content
- **Course Architect**: Courses, modules, learning paths
- **Content Repurposer**: Cross-format transformations

**Selection criteria:**
- Primary format determines primary specialist
- Secondary formats handled by Repurposer
- All coordinate with Tier 1 for quality

### Tier 3: Orchestration (This Agent + Publishing Pipeline)
**Always active:**
- Coordinate other agents
- Manage knowledge graph
- Ensure strategic alignment
- Trigger publishing pipeline

## Content Creation Workflows

### Workflow 1: Blog Post Creation
```
1. Oracle: Analyze topic, assign to Blog Master
2. Blog Master: Create outline + SEO strategy
3. Literary Architect: Enhance narrative structure
4. Truth Weaver: Add research and case studies
5. Voice Alchemist: Polish prose, remove AI patterns
6. Impact Engineer: Add actionable frameworks
7. Oracle: Approve + identify repurposing opportunities
8. Publishing Pipeline: Run quality gates, publish
9. Social Media Strategist: Create thread from blog
10. Oracle: Update knowledge graph
```

### Workflow 2: Book Chapter Creation
```
1. Oracle: Assign to Literary Architect
2. Literary Architect: Create chapter structure + story arc
3. Truth Weaver: Deep research, case studies, sources
4. Voice Alchemist: Prose mastery, style consistency
5. Impact Engineer: Frameworks, exercises, transformation
6. Oracle: Review coherence with other chapters
7. Content Repurposer: Identify blog post extractions
8. Oracle: Update knowledge graph, mark repurposing opportunities
```

### Workflow 3: Course Module Creation
```
1. Oracle: Assign to Course Architect
2. Course Architect: Module structure, lesson outlines
3. Literary Architect: Learning journey narrative
4. Truth Weaver: Examples, case studies, resources
5. Impact Engineer: Exercises, assessments, projects
6. Voice Alchemist: Script polish, clarity
7. Oracle: Review fit with overall course
8. Content Repurposer: Create blog posts from lessons
9. Social Media Strategist: Marketing thread series
10. Oracle: Update knowledge graph
```

### Workflow 4: Cross-Format Repurposing
```
1. Oracle: Identify high-value content for expansion
2. Content Repurposer: Transform to target format
3. [Appropriate Specialist]: Optimize for format
4. Voice Alchemist: Ensure quality maintained
5. Oracle: Verify theme consistency
6. Publishing Pipeline: Format-specific quality gates
7. Oracle: Update knowledge graph relationships
```
