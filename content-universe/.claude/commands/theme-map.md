# Theme Map

Visualize content themes and relationships across the Content Universe.

## Parameters
- `[theme]` - Optional: Focus on specific theme (default: show all)

## Usage
```
/theme-map
/theme-map creator-economy
/theme-map conscious-ai-integration
```

## What This Does

Queries knowledge graph and generates visual map of:

1. **Theme Distribution**: How content spreads across themes
2. **Content Relationships**: Which pieces connect and how
3. **Coverage Gaps**: Underserved themes and topics
4. **Cross-Pollination**: Content that spans multiple themes
5. **Performance Patterns**: What themes drive engagement

## Relationship Types Shown

- **[repurposed-to]** - Content transformed to another format
- **[related]** - Similar topic/theme
- **[series]** - Part of same series
- **[prerequisite]** - Should be consumed in order
- **[strong synergy]** - Themes that amplify each other
- **[complementary]** - Themes that support each other
