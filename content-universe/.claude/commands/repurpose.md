# Repurpose

Transform content from one format to another, maximizing value.

## Parameters
- `<source-file>` - Path to source content (relative to content-universe/)
- `<target-format>` - Desired output format

## Target Formats
- `chapter` - Expand to book chapter (3000-5000 words)
- `blog` - Condense/adapt to blog post (1500-2000 words)
- `thread` - Extract Twitter/X thread (8-12 tweets)
- `linkedin` - Create LinkedIn post (1300 words)
- `course-module` - Transform to course module (with lessons/exercises)
- `email-sequence` - Create email drip sequence (5-7 emails)
- `social-quotes` - Extract shareable insights (5-7 quotes)

## Usage
```
/repurpose blogs/conscious-ai/ai-ethics-creators.md chapter
/repurpose books/year-of-creator/chapter-5.md thread
/repurpose courses/ai-music-academy/module-2/lesson-3.md blog
```

## What This Does

Intelligently transforms content across formats:

1. **Content Repurposer** analyzes source and plans transformation
2. **[Format Specialist]** optimizes for target platform:
   - Blog → Chapter: Literary Architect expands narrative
   - Chapter → Blog: Blog Master condenses and SEO-optimizes
   - Any → Thread: Social Media Strategist creates hooks
   - Any → Course: Course Architect adds interactivity
3. **Voice Alchemist** maintains voice consistency
4. **Publishing Pipeline** runs format-specific quality gates
5. **Knowledge Graph** tracks repurposing relationship

## Transformation Examples

### Blog → Chapter
- 2-3x expansion (1500 → 4000 words)
- Adds narrative arc and story elements
- Deepens research and case studies
- Removes SEO optimization
- Adds frameworks and exercises

### Chapter → Blog
- Condenses to core insights
- Adds SEO keywords and structure
- Creates skimmable format (H2/H3s)
- Adds compelling hook for web
- Includes CTA to book

### Any → Thread
- Extracts 8-12 key insights
- Creates scroll-stopping hook
- Numbers tweets for thread flow
- Adds engagement question
- Optimizes for shareability

### Course Module → Blog Series
- 1 blog post per lesson
- SEO-optimized for discoverability
- Actionable excerpts
- CTAs drive to course enrollment

## Output

You'll receive:
- **Transformed content** (ready to publish)
- **Transformation notes** (what changed and why)
- **Quality report** (format-specific checks passed)
- **Related opportunities** (other repurposing ideas)
- **Knowledge graph update** (tracks relationship)
- **Performance prediction** (expected engagement)

## What Happens Next

Repurposed content will be saved to appropriate directory:
- Chapters: `content-universe/books/<book>/`
- Blogs: `content-universe/blogs/<pillar>/`
- Threads: `content-universe/social/threads/`
- Courses: `content-universe/courses/<course>/`

And tracked in knowledge graph with relationship to source.
