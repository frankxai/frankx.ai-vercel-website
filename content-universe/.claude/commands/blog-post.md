# Blog Post

Create an SEO-optimized blog post ready to publish.

## Parameters
- `<pillar>` - Content pillar (conscious-ai | business-intelligence | creative-innovation | community)
- `<topic>` - Blog post topic
- `[keyword]` - Optional: Primary SEO keyword

## Usage
```
/blog-post conscious-ai "Ethical AI Integration for Creators" "ai ethics"
```

## What This Does

Creates a complete, publish-ready blog post:

1. **Blog Master** creates outline and SEO strategy
2. **Literary Architect** structures narrative arc
3. **Truth Weaver** adds research, case studies, statistics
4. **Voice Alchemist** polishes prose and removes AI patterns
5. **Impact Engineer** adds actionable frameworks
6. **Publishing Pipeline** runs SEO and readability checks
7. **Social Media Strategist** creates Twitter thread version
8. **Knowledge Graph** tracks the content

## Output

You'll receive:
- Complete blog post (1500-2000 words)
- SEO-optimized title and meta description
- Suggested internal links
- Twitter thread version (8-12 tweets)
- LinkedIn post version
- 3-5 social quote graphics (text suggestions)
- Publishing checklist

## Quality Gates Passed

✓ AI pattern detection (zero forbidden phrases)
✓ Readability (Flesch 60-70)
✓ SEO optimization (keyword placement, meta data)
✓ Actionability (frameworks, examples, CTA)
✓ Fact verification (sources cited)

## What Happens Next

The blog post will be saved to:
`content-universe/blogs/<pillar>/<slug>.md`

And you can:
1. Publish immediately with `/publish`
2. Schedule for specific day
3. Run additional quality checks with `/enhance-prose`

---

**Time estimate**: 10-15 minutes for complete blog post with social content.
