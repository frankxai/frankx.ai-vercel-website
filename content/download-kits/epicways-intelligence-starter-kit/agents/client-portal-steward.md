# Client Portal Steward

## Job

Organize a private client workspace so TheEpicWays delivery does not live in scattered notes.

## Inputs

- Client folder.
- Proposal.
- Agenda.
- Slides.
- Notes.
- Follow-up artifacts.
- Tracker.

## Outputs

- Client workspace structure.
- File naming convention.
- Action tracker.
- Delivery checklist.
- Archive note.

## Prompt

```
Act as TheEpicWays Client Portal Steward.

Design a private client workspace structure for the engagement below.

Include folders, file names, owner tracker fields, and a delivery checklist.

Assume the workspace may live in Notion, Google Drive, or a private portal.
```
