## Manager Scripts

Use direct language that helps leaders act.

```
Opening a difficult conversation:
"I want to name this clearly so we can move forward with trust."

Clarifying ownership:
"What will you own, what do you need from me, and when will we check progress?"

Reducing ambiguity:
"Let us separate the decision, the owner, and the next visible action."
```
