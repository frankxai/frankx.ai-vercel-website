# Ana AI Business Kit

Version: 0.1.0  
Release date: 2026-06-24

This kit helps Ana turn her HR, psychology, faith, embodiment, self-knowledge, and AI companion work into a simple AI-supported business system.

It is not a public biography, a finished brand, or a promise of results. It is a working folder Ana can use to clarify offers, run sessions and circles, publish carefully, and build travel-friendly income over time.

## Start here

1. Read `01-offer-map.md`.
2. Choose one offer to run next.
3. Use the matching file:
   - `02-personal-clarity-session.md`
   - `03-guided-reflection-circle.md`
   - `04-self-knowledge-workshop.md`
4. Use `07-client-letter-and-aftercare.md` after each paid session, circle, or workshop.
5. Use the agents only after the offer is clear.

## What this includes

- A practical offer map.
- Session, circle, and workshop structures.
- A content library system.
- A 12 month freedom engine roadmap.
- A client letter and aftercare workflow.
- Eight markdown agent briefs.
- Optional ACOS, Codex, and Claude installation guidance.
- Privacy and boundary rules.

## How to use AI safely here

AI should help backstage with structure, notes, drafts, research organization, and follow-up ideas. Ana stays in front. Do not let AI pretend to be Ana, diagnose clients, replace professional care, or publish private client material.

## Public source baseline

Use Ana's current website, Work With Me page, Companions page, Research hub, and public professional profiles as the starting point. Anything deeper should wait for Ana's review and approval.
