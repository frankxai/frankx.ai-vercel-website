# ACOS, Codex, And Claude Install Guide

This kit works as plain markdown. Start here before installing anything bigger.

## Option 1 - Simple Claude or ChatGPT use

1. Create a private project or folder.
2. Upload the kit files.
3. Start with `01-offer-map.md`.
4. Paste one agent file when you need that agent.
5. Do not paste private client material unless the workspace is private and consent is clear.

## Option 2 - Codex workspace

1. Create a private repository or folder for Ana.
2. Copy this kit into `/docs/ana-ai-business-kit`.
3. Ask Codex to help maintain the files, generate worksheets, or check public copy against `privacy-boundaries.md`.
4. Keep client notes out of git unless the repo is private and intentionally secured.

## Option 3 - ACOS later

ACOS is optional power tooling. It is not the product.

Use ACOS when Ana wants Codex or Claude to help run repeated workflows:

- offer review
- content library maintenance
- workshop drafting
- public copy checks
- monthly business review

FrankX download page:

`/downloads/preview/agentic-creator-os`

## Recommended v1 stance

Do not create a Codex plugin yet. Validate the markdown agents and workflows first. Build a plugin only after Ana knows which workflows she actually repeats.
