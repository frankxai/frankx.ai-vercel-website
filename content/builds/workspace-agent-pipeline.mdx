## What we built

A pipeline that takes a content brief in a Google Doc, researches via grounded Gemini calls, drafts the article in the same Doc, schedules outreach in Gmail, and logs results to a Sheet. Runs as Managed Agents in persistent isolated environments, so jobs continue across browser sessions.

## The stack and why

- **Gemini API Managed Agents** — persistent isolated environments; jobs survive browser restarts.
- **Google Workspace** — the surface where the work lives; no extra UI to learn.
- **Claude Code** — built the Apps Script bridge.

## What worked

- The "do the work where the user already is" thesis held — zero context switching during the demo.

## What broke

- OAuth scope sprawl: the agent needed five separate scopes; consolidation pass pending.
