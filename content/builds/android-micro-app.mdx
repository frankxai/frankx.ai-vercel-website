## What we built

A focused single-purpose Android app generated from one prompt in Antigravity 2.0. The brief: a daily prompt-of-the-day app that pulls from the FrankX prompt library API. Tests how far agentic mobile development goes without manual code.

## The stack and why

- **Antigravity 2.0** — native Android target.
- **Gemini API** — content layer.

## What worked

- The scaffold compiled on the first attempt.

## What broke

- The agent did not initially handle the Android lifecycle correctly — needed a follow-up prompt to add proper state persistence across rotations.
