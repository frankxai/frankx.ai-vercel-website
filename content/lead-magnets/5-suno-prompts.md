## 1. The Awakening (142 plays, 20 likes)

**Genre**: African / World / Cinematic

### The Prompt

```

natural, world music, tribal drums, layered female choir, call and response,
dramatic build, soul-stirring, rhythmic percussion, organic, raw energy
```

### Why It Works

- **"10 female vocalists"** -- Suno interprets this as dense layered vocals, creating a rich choral texture
- **"Call and response"** -- triggers an antiphonal structure (lead voice, then group response)
- **"Dramatic build"** -- signals song structure: quiet start â†’ powerful climax
- **"Tribal drums"** -- produces organic percussion patterns instead of electronic beats

### Variations to Try

- Swap "African" for "Mongolian" or "Celtic" for different world music flavors
- Replace "10 female vocalists" with "male baritone choir" for a deeper register
- Add "432Hz" or "528Hz" for healing frequency tuning

## 2. Vibe O S (128 plays, 24 likes)

**Genre**: Hip Hop / Bass

### The Prompt

```
Female hip hop, bass-heavy, lyrical, sophisticated, violin, electric guitar,
808 sub bass, confident female vocalist, clever wordplay, cinematic strings,
modern trap production, polished, radio-ready, 4 minutes
```

### Why It Works

- **"Sophisticated"** -- one of the most powerful Suno keywords. It elevates vocabulary, melody complexity, and production quality
- **"Violin" + "electric guitar"** -- hybrid instrumentation creates a unique sound that blends classical with modern
- **"Bass-heavy" + "808 sub bass"** -- double emphasis ensures deep low-end presence
- **"4 minutes"** -- explicitly requesting longer duration (default is ~2 min)

### Variations to Try

- Replace "female" with "male" or "duet" for different vocal arrangements
- Add "jazz chords" or "neo-soul" for a smoother feel
- Swap "trap production" for "boom bap" for classic hip hop texture

## 3. Golden Age of Intelligence (119 plays, 20 likes)

**Genre**: EDM / Metalcore / Hip Hop

### The Prompt

```
EDM hit, metalcore, sophisticated hip hop, female duet, electric guitar,
festival anthem, massive drop, stadium energy, heavy breakdown, soaring
vocals, electronic meets metal, high production value, radio edit
```

### Why It Works

- **Genre fusion** -- combining 3 genres ("EDM hit, metalcore, sophisticated hip hop") forces Suno into creative territory. Most producers use one genre. Fusing three creates something original.
- **"Female duet"** -- produces vocal interplay: harmonies, call-and-response, and complementary ranges
- **"Massive drop" + "heavy breakdown"** -- structural keywords that give the song a clear peak moment
- **"Festival anthem" + "stadium energy"** -- these context words set the scale and ambition of the production

### Variations to Try

- Replace "metalcore" with "orchestral" for an epic cinematic crossover

## Prompt Engineering Cheat Sheet

### Power Words (use these to level up any prompt)

| Category       | Keywords                                                                  |
| -------------- | ------------------------------------------------------------------------- |
| **Quality**    | sophisticated, polished, radio-ready, high production value, remastered   |
| **Emotion**    | dramatic, uplifting, haunting, euphoric, melancholic, triumphant          |
| **Structure**  | build, drop, breakdown, bridge, outro, verse-chorus-verse                 |
| **Vocals**     | belting, head voice, whisper, choir, layered harmonies, call and response |
| **Production** | Dolby Atmos, THX, cinematic, spatial, wide stereo                         |

### Common Mistakes

1. **Too vague**: "Make a good song" -- gives Suno no direction
2. **Too short**: Single-word genres produce generic results. Stack descriptors.
3. **Contradictions**: "Calm, aggressive" confuses the model. Pick one emotional direction per section.
4. **Ignoring duration**: Default is ~2 min. Add "3 minutes" or "4 minutes" for longer tracks.
