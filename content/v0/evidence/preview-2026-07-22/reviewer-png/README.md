# Cross-Provider Visual Review Packet

These PNGs are lossless transport derivatives of the canonical WebP receipts in the parent directory.

- Catalog and package pages are cropped from the top to the tested viewport height: 900 px desktop and 844 px mobile.
- Studio captures already match the tested viewport and were converted without cropping.
- No interface content, color, spacing, or asset was edited.
- The packet exists because the v0 review transport accepted PNG but did not reliably decode WebP.

The parent `report.json` remains the machine-readable QA source of truth.
