## Key Features

### 1. Multi-Agent Orchestration
The brain that coordinates all agents, analyzes requests, and aggregates results.

### 2. Specialized Agents
- **Writer Agent**: Blog posts, articles, social media, video scripts
- **Researcher Agent**: Fact-checking, finding sources, context
- **Publisher Agent**: Platform optimization, SEO, formatting

### 3. Real-Time Dashboard
Built with Next.js 16 + React 19 with live streaming updates.

### 4. VS Code Extension
Bring Creator OS directly into your IDE.

- **Next.js 16** with App Router

- **VS Code Extension API**
- **Tailwind CSS** + shadcn/ui

## Getting Started

```bash
# Clone the repository
git clone https://github.com/frankx-org/agentic-creator-os.git
cd agentic-creator-os

# Install dependencies
npm install

# Set up database
npx prisma generate
npx prisma db push

# Start development server
npm run dev
```
