## FrankX / ACOS Workflow Recommendations (Hybrid That Ships)

**Native first (this harness + Grok tools)**: image_gen for heroes and infogenius-style visuals, image_to_video + reference_to_video for quick clips and music video keyframes. Zero extra cost, immediate iteration, full control in the session. Use for 70-80% of volume and testing.

**Premium cinematic when it earns its keep**: Higgsfield (or direct Kling / Veo access) for hero sequences, character-consistent series, native audio sync, or when you need the full menu + Soul ID + camera control in one place. Wire the MCP where available (Claude Code examples are real and powerful for end-to-end "plan → generate → grade → ship").

**Research layer (hooks + scripts)**: Sandcastles.ai (or equivalent X-derived outlier analysis) for short-form. It surfaces what is actually working in your niche right now and turns it into data-backed hooks and full scripts. Pair it with the generation layer instead of guessing. This is the "research + generation flywheel."

**Taste lanes & gate (non-negotiable)**: Hold every output against one lane (cinematic, noir-tech, studio-organic, etc.). Never mix lanes on one asset. Run the gate: storytelling first, then technical, then brand voice. Edit rhythm, pacing, and human framing turn "AI video" into "the thing that stops the scroll."

**Learning loop**: Log what you ship, which model + lane + prompt pattern won, what the audience actually watched. Feed it back into the next brief. This is how the menu gets smarter than any single vendor.

## FAQ

**Is Higgsfield the single best tool?**  
No single tool is. It is currently the strongest access layer + integration story for many prosumers (one sub, many models, MCP, character tools). Route the best model for the shot.

**Do I need Sandcastles.ai?**  
Only if short-form research and hook writing is a bottleneck. It is excellent at turning real viral data into scripts. Many creators get similar signals from manual X/IG research + their own hook learning systems.

**What about affiliate income?**  
Higgsfield Earn (performance) and strong recurring programs like ElevenLabs are the ones creators talk about actually paying. Apply to the ones you use. Disclose. Only recommend what you ship.

**How do I avoid slop?**  
Study real directors and editors (pacing, framing, attention). Generate with intention (one lane, strong references, story first). Edit ruthlessly. The tool is only half the pipeline.

**Will this post be updated?**  
Yes. The frontier moves monthly. Check the date and the "Updated" notes on related posts. The menu logic stays; the specific model rankings rotate.
