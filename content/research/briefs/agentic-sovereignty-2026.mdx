# Agentic Sovereignty — Research Brief

**TL;DR**: Sovereignty is fake when you cannot export, delete, rehost, or audit memory and agent traces. Production pattern: **local_core authority** + scored adapters + privacy-class routing. This is failure mode #3 of agentic life infrastructure.

## Embedding Keywords

`agentic sovereignty`, `local_core authority`, `exportable memory atoms`, `agent trace export`, `privacy class routing`, `no silent training`, `dual-write memory policy`, `self-host agent memory`, `public private content gates`, `data portability AI agents`, `GDPR AI memory`, `EU AI Act oversight`, `open-core agent OS`

## Four Rights

1. **Export** — open formats for memory, traces, skills, receipts  
2. **Delete** — hard delete with proof  
3. **Rehost** — move without losing IDs/provenance  
4. **Audit** — actor-aware who/when/what  

## Architecture

```
local_core (authority)
  → privacy-class router
  → adapters (Mem0 / Graphiti / Letta / cloud)
  → harnesses (Claude / Codex / Hermes / …)
```
