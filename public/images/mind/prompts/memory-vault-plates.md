## CONCEPT
Memory as organized plates: the operator carries reusable context, not random notes.

## ART DIRECTION
Premium FrankX system archive. Smoked glass layers, graphite indexing rails, subtle emerald and cyan routing marks, documentary precision.

## SCENE
A set of translucent glass memory plates suspended above a black metal tray. Each plate contains abstract etched routes and faint blocks with no readable writing. The plates align into a clean stack, like a private context archive.

## COMPOSITION
3:2 crop. Layered depth from foreground tray to background plates. The stack sits slightly right of center with quiet negative space at left.

## LIGHTING
Soft top light, cyan edge light, emerald micro pulses along two routes, black background falloff, crisp reflections.

## PALETTE
Obsidian, graphite, smoked glass, pale silver, emerald #10b981, cyan #06b6d4.

## MOOD
Order, recall, trust, and care. It should feel like a memory system built by an engineer, not a mystical archive.

## CONSTRAINTS
No readable text, no logos, no hands, no faces, no brain, no book fantasy, no medical scan, no stock database icon, no floating particles. Ready for immediate critic.
