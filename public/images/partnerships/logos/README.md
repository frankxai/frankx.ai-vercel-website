## Replacing with official press-kit assets

For any partner conversation that crosses into formal contract, swap the
Simple-Icons / Wikipedia-sourced logo for the official press-kit asset:

- **Arrow** — https://www.arrow.com/en/about-arrow/press-room
- **NVIDIA** — https://nvidianews.nvidia.com/ then Brand Assets
- **Anthropic** — https://www.anthropic.com/brand
- **Vercel** — https://vercel.com/design/brand
- **Google** — https://about.google/brand-resource-center/

When swapping, update this README's row with the new source URL and license
note. Keep the per-slug filename so partner configs don't need to change.
