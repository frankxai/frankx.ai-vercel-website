# 08 - 12 Week Startup Roadmap

## Weeks 1-2 - Clarify

- Confirm the first audience.
- Choose one offer.
- Write the one-page operating map.
- Set up the private workspace.
- Publish the first founder note and carousel.

## Weeks 3-4 - First Proof

- Run 2 to 3 conversations.
- Test the clarity session or local lab offer.
- Capture objections and useful phrases.
- Publish a build log.
- Decide what should become a template.

## Weeks 5-6 - Local Lab

- Install only the tooling that supports the first offer.
- Create repo structure.
- Add ACOS packs if useful.
- Add Codex and Claude instructions.
- Draft the first OpenClaw or voice routine.

## Weeks 7-8 - Media Engine

- Produce a repeatable carousel style.
- Record short video scripts.
- Draft first podcast outline.
- Build content library tags.
- Run weekly review.

## Weeks 9-10 - Productize

- Turn the most repeated session into a paid template or workshop.
- Create simple delivery checklist.
- Write onboarding and aftercare.
- Add proof capture.
- Update pricing and positioning.

## Weeks 11-12 - Decide

- Review what created demand.
- Review what drained energy.
- Keep one offer.
- Kill one distraction.
- Choose the next build sprint.
