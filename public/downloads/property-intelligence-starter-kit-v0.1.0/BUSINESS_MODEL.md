# Business Model

Property Intelligence OS should be sold as both a free template and a premium implementation business.

## Free Community Layer

Purpose:

- earn trust
- teach the operating model
- let builders fork and improve the system
- create demand for private installs

Includes:

- public starter kit
- Vercel/v0 prompt
- install runbook
- agent team map
- MCP architecture
- data boundaries
- success gates

## Paid Owner Install

Typical setup: 1,500 to 7,500 EUR.

Scope:

- property intake
- approved facts and FAQ
- premium property page
- renter portal
- inquiry/support forms
- listing draft studio
- owner dashboard
- first agent dry-runs
- handoff training

## Managed Operations

Typical retainer: 300 to 2,000 EUR per month.

Scope:

- weekly owner review
- FAQ improvements
- listing refresh
- inquiry quality review
- maintenance routing hygiene
- privacy checks
- analytics and proof collection
- integration backlog management

## Agency / Partner License

Purpose:

- let implementers sell this to owners and agencies
- standardize quality
- preserve safety boundaries
- create recurring revenue around setup and operations

Partner package:

- white-label install checklist
- proposal template
- quality gates
- privacy scan workflow
- Vercel preview gate
- Railway MCP runbook
- proof dashboard

## Proof Metrics

- fewer repeated owner messages
- faster inquiry first draft
- better listing completeness
- higher FAQ coverage
- clearer urgent escalation
- no private data in public artifacts
- premium portal screenshot quality
