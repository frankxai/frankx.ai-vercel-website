# Download And Fork Guide

## Community Path

Use this kit for learning, local experimentation, and public-safe community forks.

1. Fork or copy `property-os-template`.
2. Fork or copy `property-portal-template`.
3. Run validation in both repos.
4. Replace only sample-safe content.
5. Keep private owner/renter data in a private install workspace.

## Partner Path

Use this kit to sell installs and managed operations.

1. Start from `property-os-template`.
2. Create a private client workspace.
3. Wire approved facts into `property-portal-template`.
4. Deploy a Vercel preview.
5. Run visual QA and release gates.
6. Offer a monthly operating retainer.

## Do Not Put In Public Forks

- renter names tied to addresses, issues, payments, or leases
- identity documents
- bank details
- access codes
- private owner financials
- lease or contract files
- unresolved private disputes
