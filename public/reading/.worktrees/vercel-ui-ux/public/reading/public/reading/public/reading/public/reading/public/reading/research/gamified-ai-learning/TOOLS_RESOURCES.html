Tools & Resources for Gamified AI Learning

Open Source Projects

1. Tutor-GPT by Plastic Labs

What: LLM-powered learning companion with Theory of Mind License: GPL-3.0 Stack: Next.js, app router, Honcho for user representations Key Feature: Dynamically updates its own prompts based on learner state GitHub: https://github.com/plastic-labs/tutor-gpt

2. OnlineMate (Stanford SCALE)

What: Multi-agent companion system for cognitive support Key Feature: Simulates peer-like agent roles, adapts to cognitive states Paper: Available via Stanford SCALE Initiative

3. Llama Tutor

What: AI personal tutor with adaptive personality Focus: Language learning companion URL: https://www.rundown.ai/tools/llama-tutor-ai-personal-tutor
