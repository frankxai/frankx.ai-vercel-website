Skill: Wallet Ops

Purpose

Perform wallet checks, balances, and safe transfer preparation.

Trigger

  • You need to read balances or initiate a transfer.

Inputs

  • Address
  • RPC URL

Outputs

  • Balance snapshot
  • Transfer plan with spend limits

Tools

  • JSON-RPC
  • Viem or ethers

Workflow

  1. Fetch native balance for the address.
  2. Check token balances if required.
  3. Build a transfer plan with max value thresholds.

Validation

  • Balance call returns a valid wei value.
  • Spend limits enforced before send.

Failure Modes

  • Incorrect address format: validate checksum.
  • Insufficient funds: halt and log.

Examples

curl -s -X POST "$RPC_URL" \
  -H 'content-type: application/json' \
  --data '{"jsonrpc":"2.0","id":1,"method":"eth_getBalance","params":["0x...","latest"]}'
