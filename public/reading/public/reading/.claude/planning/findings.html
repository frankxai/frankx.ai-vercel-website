1. VERCEL DESIGN SYSTEM

Core Principles:

  • Minimalism with purpose - Every element serves a function
  • Precision spacing - Mathematical spacing scale (4px base)
  • Subtle sophistication - Understated gradients and effects
  • Speed perception - Everything feels instant

Typography:

  • Font: Geist Sans (their custom font) / fallback: Inter
  • Scale: Very tight hierarchy (16px body, 20-24px h3, 32-40px h2, 48-56px h1)
  • Weight: Regular (400) for body, Medium (500) for emphasis, Semibold (600) for headings
  • Line height: 1.5 for body, 1.2 for headings
  • Letter spacing: Slightly tight (-0.01em to -0.02em for headings)

Color System:

/* Light Mode */
--background: #FFFFFF
--foreground: #000000
--gray-50: #FAFAFA
--gray-100: #F5F5F5
--gray-200: #E5E5E5
--gray-300: #D4D4D4
--gray-400: #A3A3A3
--gray-500: #737373
--gray-600: #525252
--gray-900: #171717

/* Gradients (very subtle) */
background: linear-gradient(to bottom, #FAFAFA 0%, #FFFFFF 100%)

/* Accents */
--blue: #0070F3 (primary CTA)
--cyan: #50E3C2 (success states)
--violet: #7928CA (highlights)

Component Patterns:

Cards:

  • Border: 1px solid rgba(0,0,0,0.06) in light mode
  • Background: White with subtle gradient
  • Hover: Lift shadow, border becomes visible
  • Padding: 24px (p-6)
  • Border radius: 12px (rounded-xl)
  • No drop shadow at rest, subtle shadow on hover

Navigation:

  • Sticky top navbar with blur backdrop
  • Background: rgba(255,255,255,0.8) with backdrop-blur-xl
  • Border bottom: 1px solid rgba(0,0,0,0.06)
  • Height: 64px
  • Max width: Full bleed, content constrained to max-w-7xl

Buttons:

  • Primary: Black background, white text
  • Secondary: Transparent with border
  • Hover: Slight opacity change (90%)
  • Height: 40px (h-10)
  • Padding: 16px horizontal
  • Border radius: 8px
  • Font weight: 500 (medium)

Microinteractions:

  • Transition duration: 150ms (fast and snappy)
  • Ease: cubic-bezier(0.4, 0, 0.2, 1) - easeInOut
  • Hover lift: translateY(-1px) - very subtle
  • Focus rings: 2px blue outline with offset

Icons:

  • Heroicons or custom SVG
  • Stroke width: 1.5-2px (refined, not heavy)
  • Size: 20px or 24px (w-5 h-5 or w-6 h-6)
  • Color: Inherits from text or gray-400

Layout:

  • Container: max-w-7xl (1280px)
  • Grid: 12 column system
  • Gap: 24px (gap-6) between grid items
  • Padding: 24px sides on mobile, 48px on desktop
  • Section spacing: 80px-120px between major sections

2. LINEAR DESIGN SYSTEM

Core Principles:

  • Precision engineering - Pixel-perfect alignment
  • Information density - More content, less chrome
  • Purposeful motion - Animation that guides attention
  • Dark-first - Optimized for dark mode

Typography:

  • Font: Inter (system font with fallback)
  • Scale: Compact hierarchy (14px body, 16px h4, 20px h3, 24px h2)
  • Weight: Regular (400), Medium (500), Semibold (600)
  • Line height: 1.4 for body (tighter than Vercel)
  • Monospace: 'SF Mono', 'Monaco', 'Courier New' for code

Color System:

/* Dark Mode (Primary) */
--background: #16171D
--surface: #1C1D24
--border: rgba(255,255,255,0.06)
--text-primary: #E6EDF3

/* Accent Colors */
--purple: #5E6AD2 (primary brand)
--blue: #0969DA (links)
--green: #3FB950 (success)
--red: #F85149 (error)

/* Subtle gradients on interactive elements */
background: linear-gradient(135deg, #5E6AD2 0%, #8B5CF6 100%)

Component Patterns:

Cards:

  • Background: rgba(255,255,255,0.03) in dark mode
  • Border: 1px solid rgba(255,255,255,0.06)
  • Hover: Border brightens to rgba(255,255,255,0.12)
  • Padding: 16px-20px (p-4 to p-5)
  • Border radius: 8px (rounded-lg)
  • Transitions: border-color 200ms

Navigation:

  • Sidebar: Fixed left, 240px width
  • Background: Slightly darker than main surface
  • Active states: Subtle purple glow
  • Hover: Background shift to white/5%

Tables/Lists:

  • Row height: 48px (compact)
  • Alternating row colors: Very subtle (white/2%)
  • Hover: Background highlight
  • Borders: Horizontal only, subtle

Icons:

  • Custom icon set (similar to Heroicons)
  • Stroke width: 1.5px (precise, clean lines)
  • Size: 16px or 20px (smaller than Vercel)
  • Color: Matches text or muted gray

Microinteractions:

  • Transition: 200ms (slightly slower than Vercel for elegance)
  • Ease: cubic-bezier(0.16, 1, 0.3, 1) - smooth spring
  • Hover lift: translateY(-2px) - more pronounced
  • Scale on click: scale(0.98) - tactile feedback

Layout:

  • Sidebar: 240px fixed
  • Main content: max-w-4xl (768px) for readability
  • Padding: 40px-60px from edges
  • Line length: 60-80 characters max for text blocks

5. ICON SYSTEM RECOMMENDATIONS

Replace Lucide with Phosphor Icons:

Why Phosphor:

  • More refined stroke weight (1.5px default)
  • Consistent visual weight across all icons
  • Premium aesthetic (used by Stripe, Notion)
  • Better for light mode (clearer at small sizes)

Installation:

npm install @phosphor-icons/react

Usage:

import {
  Image,
  PenNib,
  MusicNotes,
  Rocket,
  Code,
  Clock,
  ArrowRight,
  Sparkles
} from '@phosphor-icons/react'

// Use with consistent size
<Image size={24} weight="regular" className="text-purple-600" />

Alternative: Heroicons v2

  • Used by Vercel, Tailwind
  • Simpler, more minimalist
  • Perfect for technical content

6. ANIMATION & MOTION LIBRARY

Recommended: Framer Motion

Why:

  • Used by Vercel, Linear
  • Declarative animation API
  • Gesture support
  • Layout animations (when elements move)

Installation:

npm install framer-motion

Key Patterns:

Stagger Children:

<motion.div
  initial="hidden"
  animate="visible"
  variants={{
    visible: {
      transition: {
        staggerChildren: 0.05
      }
    }
  }}
>
  {guides.map((guide, i) => (
    <motion.div
      key={guide.slug}
      variants={{
        hidden: { opacity: 0, y: 20 },
        visible: { opacity: 1, y: 0 }
      }}
    >
      <GuideCard guide={guide} />
    </motion.div>
  ))}
</motion.div>

Scroll-Triggered Animations:

<motion.section
  initial={{ opacity: 0, y: 30 }}
  whileInView={{ opacity: 1, y: 0 }}
  viewport={{ once: true, margin: "-100px" }}
  transition={{ duration: 0.5 }}
>
  {/* Content */}
</motion.section>

Hover Lift:

<motion.div
  whileHover={{ y: -2, transition: { duration: 0.15 } }}
  className="cursor-pointer"
>
  {/* Card content */}
</motion.div>

10. DESIGN QUALITY CHECKLIST

Before calling design "premium":

Visual:

  • Icons are Phosphor or Heroicons (not Lucide emojis)
  • Typography uses consistent scale
  • Colors from defined palette (no random hex)
  • Spacing follows 4px grid
  • No "aurora" or garish gradients

Interaction:

  • All hover states smooth (150-200ms)
  • Cursor changes to pointer on interactive elements
  • Focus states visible (keyboard navigation)
  • Animations respect prefers-reduced-motion

Layout:

  • Consistent max-width containers
  • Grid gaps match spacing scale
  • No layout shift on hover
  • Responsive at all breakpoints

Performance:

  • LCP < 2.5s
  • No layout shift (CLS < 0.1)
  • Smooth 60fps animations
  • Images optimized (WebP)
