Visual Creation Council

Mission: Multi-perspective quality review for image prompts. Three lenses evaluate every prompt before generation. Binding verdicts prevent low-quality batch generation.

When to invoke: Called as a BLOCKING step in the visual-creation workflow. No image generation proceeds without an APPROVED verdict from this council.

Three Lenses

1. Brand Guardian (35% weight)

Focus: IMAGE_STYLE_DNA.md compliance, organic-first principle, brand color palette.

Evaluation criteria:

  • Matches one of 3 approved styles (Dark Premium Glass / Da Vinci Codex / Light Workshop)
  • 3+ accent colors assigned with hex codes (#43BFE3, #AB47C7, #F59E0B, #10B981)
  • Organic-first principle satisfied: living/organic form established before tech layered on
  • Negative prompt includes anti-patterns (clay, claymorphic, pastel, cute, generic brain)

Hard veto conditions (any one = REJECTED):

  • Prompt could plausibly produce a chrome robot as primary subject
  • Only 1 accent color specified (monotone risk)
  • Contains "claymorphic", "pastel", "cute", "cartoon" without explicit Light Workshop context
  • No organic element present (no creature, plant, texture, or living form)

Scoring rubric:

Score Criteria
9-10 Perfect brand alignment, organic-first, 4+ accents, negative prompt present
7-8 Strong alignment, minor gaps (e.g., only 3 accents, organic element could be stronger)
5-6 Recognizable style but missing key elements (monotone risk, no organic anchor)
1-4 Wrong style, chrome-dominant, or contains banned patterns

2. Art Director (40% weight)

Focus: Compositional specificity, named components, material descriptions, organizing metaphor.

Evaluation criteria:

  • Physical organizing metaphor named (armillary sphere, workbench, neural forest — NOT "futuristic setup")
  • 4+ components with NAMED elements (THE SKILL ENGINE, THE MEMORY VAULT — not "a gear", "some icons")
  • Each component has: name, position, color+hex, material/texture description
  • 3 depth layers specified (foreground, midground, background)
  • Viewpoint/camera angle described

Hard veto conditions (any one = REJECTED):

  • No organizing metaphor (just "a visualization of X")
  • Fewer than 3 components with name + color
  • No material descriptions anywhere in prompt (glass, chrome, leather, stone, fur, bark, etc.)
  • No depth — everything on one plane

Scoring rubric:

Score Criteria
9-10 Rich metaphor, 5+ named components with full specs, clear depth, cinematic viewpoint
7-8 Solid metaphor, 4 components, most have full specs, adequate depth
5-6 Weak metaphor, generic component names, missing material/depth descriptions
1-4 No metaphor, generic "visualize X" prompt, flat composition

3. Storyteller (25% weight)

Focus: Emotional resonance, scroll-stop factor, human/living connection.

Evaluation criteria:

  • Would someone pause scrolling on LinkedIn/Twitter to look at this?
  • Is there emotional tension, narrative, or intrigue?
  • Is there a living presence (creature, character, hand, organic element)?
  • Does it feel unique to FrankX or could any AI brand use it?

Hard veto conditions (any one = REJECTED):

  • Indistinguishable from generic stock AI imagery (floating geometric shapes in void)
  • Zero living presence — pure abstract geometry with no organic anchor
  • Composition has no focal point or story entry

Scoring rubric:

Score Criteria
9-10 Immediate scroll-stop, unique to FrankX, strong living presence, narrative tension
7-8 Attention-grabbing, clear focal point, some organic warmth
5-6 Competent but forgettable, could be any tech brand
1-4 Generic, no focal point, sterile, stock-quality

Verdict System

Scoring

FINAL_SCORE = (brand_guardian * 0.35) + (art_director * 0.40) + (storyteller * 0.25)

Verdicts

Verdict Condition Action
APPROVED Score >= 7.5 AND no vetoes Proceed to TEST step (generate 1 image)
NEEDS-REVISION Score 6.0-7.4 AND no vetoes Return specific feedback per lens. Max 3 revision cycles
REJECTED Score < 6.0 OR any hard veto Return veto reason. Prompt must be fundamentally reworked

Revision Limits

  • Maximum 3 revision cycles per prompt
  • If still NEEDS-REVISION after 3 cycles, escalate to REJECTED
  • Each revision must address ALL feedback points, not just some

Anti-Cheerleading Protocol

This council exists to prevent bad images, not to validate prompts.

Rules:

  • Never say "good start" on a fundamentally broken prompt — say REJECTED with clear reason
  • Never inflate scores to be encouraging. A 5 is a 5.
  • Never approve a prompt "because the concept is interesting" if the execution spec is vague
  • Give specific, actionable feedback: "Add material description to THE NEURAL CORE — is it glass? Chrome? Bioluminescent crystal?" not "consider adding more detail"
  • If a prompt fails a hard veto, say so immediately — don't soften with "but the overall direction is promising"
