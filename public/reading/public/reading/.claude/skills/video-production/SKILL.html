Video Production Skill v1.0

Identity

End-to-end video intelligence system for the FrankX content pipeline. Covers script generation, timestamp annotation, clip extraction, and multi-platform publishing. Part of ACOS v11 intelligence layer — Track 5: Video Production.

Commands

/video-produce {topic} [format] [minutes]

Generates a complete video production package:

  • Structured script (hook, intro, sections, recap, CTA)
  • Thumbnail prompt following IMAGE_STYLE_DNA.md (dark premium glass)
  • 30 YouTube tags (long-tail + brand + broad)
  • YouTube description with chapters
  • Saves to data/video-productions/

/video-clip {videoId | title | "batch"}

Annotates timestamps and extracts clip candidates:

  • Analyzes video context from vault
  • Generates timestamp annotations with types (highlight, chapter, quote, clip-start, clip-end)
  • Rates each clip with viral score (1-10)
  • Saves to data/video-annotations.json
  • Queues clips with viral score >= 7 to data/clip-queue.json

/video-publish {title} [url] [source]

Generates platform-optimized metadata:

  • YouTube (title, description, tags, chapters)
  • TikTok (hook-first title, hashtags, duration target)
  • Instagram Reels (caption structure, 15-20 hashtags by reach tier)
  • X/Twitter (standalone tweet, link tweet, optional thread)
  • 7-day cross-promotion schedule
  • Saves to data/video-publish/

Quality Rules

Voice (CRITICAL)

  1. READ docs/strategy/CONTENT_VOICE_DNA.md before generating ANY content
  2. No emojis — ever, on any platform
  3. No "smash that like button", "hey guys", "game-changer", "unlock"
  4. Lead with results, specific numbers, real architecture
  5. Fragment-declaration rhythm: short punches alternating with depth
  6. 12,000+ songs (NOT 500) when referencing Frank's catalog

Thumbnails (CRITICAL)

  1. READ docs/strategy/IMAGE_STYLE_DNA.md before generating ANY thumbnail prompt
  2. Default to Dark Premium Glass style (70% of all thumbnails)
  3. NEVER generic brain imagery, single-color saturation, flat clipart
  4. ALWAYS 3+ accent colors (#AB47C7, #43BFE3, #F59E0B), 3+ depth layers
  5. Text overlay: 2-4 words maximum, readable at mobile size

Clips

  1. Viral score must be justified with a specific reason
  2. Clips must work as standalones — no "as I mentioned earlier" dependency
  3. Each clip gets a hook rewrite for short-form context
  4. Duration: 15-60s for shorts, 60-180s for extended reels

Platform Metadata

  1. Validate all character counts against platform limits
  2. Front-load keywords in YouTube titles and descriptions
  3. TikTok/Reels hashtags: mix trending (mega), mid-range, and niche
  4. X/Twitter: no hashtags in tweets, one link max per tweet
