﻿# /arcanea — Arcanean World Command v2.0

The world of Arcanea is a forkable creative civilization OS. Context compounds. Canon is protected. Every session grows the world.

WorldState Protocol (Always First)

Before generating anything, load:

  • data/arcanea-world-state.json → check canon.established, world_threads.open, lore_episodes, eldrian_registry[*].episodes_generated

No lore is generated that repeats an existing episode or contradicts established canon.

Input Modes

/arcanea lore [eldrian]

Generate a new lore episode for the specified Eldrian.

Process:

  1. Load WorldState — check eldrian_registry.[Eldrian].episodes_generated and world_threads.open
  2. Identify which world thread this episode will advance
  3. Generate lore prose (150-250 words) in the Eldrian's locked voice
  4. Format into 4-tweet X thread (280 chars max each, zero hashtags)
  5. Tweet 3 must connect mythology to present reality
  6. Save to data/arcanea-content-queue.json as new entry
  7. Append episode to data/arcanea-world-state.json lore_episodes array
  8. Update eldrian_registry.[Eldrian].episodes_generated count
  9. Mark any new canon under_construction in WorldState
  10. Run quality gate before saving (see below)

/arcanea week

  • Monday: Aethelin — Origin or Prophecy
  • Tuesday: Korghast — Revelation
  • Wednesday: Velmara — Riddle
  • Thursday: Solrex — Revelation
  • Friday: Zyranthis — Entropy meditation
  • Saturday: Image post (strongest visual Eldrian this week)
  • Sunday: THE CONCLAVE — all five speak on one truth

Saves all 7 pieces to content queue. Updates WorldState. Notifies Slack #content.
