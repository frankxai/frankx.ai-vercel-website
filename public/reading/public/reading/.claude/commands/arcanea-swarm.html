/arcanea-swarm — Arcanean Multi-Agent Content Engine

Purpose: Deploy a coordinated agent swarm to generate Arcanean content at scale — a full week of X threads, all 5 Eldrian character cards, or a complete narrative arc.

Input

Provide ONE of:

  • week — generate 7 days of Arcanean X content (7 threads + 2 images)
  • characters — generate all 5 Eldrian character images + companion lore
  • arc [name] — generate a 4-episode narrative arc:
    • arc formation — how Arcanea was built
    • arc schism — Aethelin vs Zyranthis philosophical war
    • arc builders — how humans learned from the Eldrians
    • arc code — Aethelin's proto-code and its AI echoes (most shareable)
  • [N] — generate N lore fragments, distributed across Eldrians
  • No input: defaults to week

Swarm Architecture

ORCHESTRATOR (Claude Code — this session)
    │
    ├── Phase 1: LORE AGENTS (parallel)
    │   ├── Generate lore for Monday Eldrian
    │   ├── Generate lore for Tuesday Eldrian
    │   ├── Generate lore for Wednesday Eldrian
    │   └── ... (all planned days)
    │
    ├── Phase 2: FORMAT AGENTS (sequential per piece)
    │   └── Format each lore into 4-tweet thread + validate voice
    │
    ├── Phase 3: IMAGE AGENTS (parallel — 2-3 featured)
    │   ├── Generate Eldrian image 1 (nano-banana)
    │   └── Generate Eldrian image 2 (nano-banana)
    │
    └── Phase 4: QUEUE AGENT
        ├── Save all to data/arcanea-content-queue.json
        ├── Merge into data/social-queue.json
        └── Send batch summary to Slack #content

Process

Step 1: Load Swarm Context

Read .claude/skills/arcanea-swarm/SKILL.md and .claude/skills/arcanea/SKILL.md

Step 2: Plan the Batch

For week mode, assign per day:

Day Eldrian Episode
Mon Aethelin Origin/Prophecy
Tue Korghast Revelation
Wed Velmara Riddle
Thu Solrex Revelation
Fri Zyranthis Entropy/Battle
Sat Featured Image post (no text thread)
Sun All Five "Conclave" — all speak

Step 3: Execute Lore Generation (parallel)

For each planned piece:

Generate [episode type] lore for [Eldrian]
→ 150-200 words of cosmic prose
→ 4-tweet X thread (per voice rules)
→ Store in memory array

Run all days in parallel where possible.

Step 4: Format + Validate

For each generated piece:

  • Format into final X thread structure
  • Check character counts (flag any >280)
  • Validate against Arcanea voice rules
  • Assign scheduled date

Step 5: Generate Images (parallel)

Pick 2 Eldrians with strongest visual potential for this batch. Generate using nano-banana MCP:

  • Aspect: 9:16 for character cards, 16:9 for scene posts
  • Save to: public/images/arcanea/eldrian-[name]-[YYYYMMDD].png

Step 6: Save Batch

Write to data/arcanea-content-queue.json:

{
  "batch_id": "arcanea-[YYYYMMDD]",
  "generated_at": "ISO timestamp",
  "pieces": [...]
}

Step 7: Slack Report

Send batch summary to #content:

ARCANEAN SWARM — [N] pieces generated

MON: Aethelin (Origin) — [char count] chars
TUE: Korghast (Revelation) — [char count] chars
...
SAT: [image post] — Velmara character card generated
SUN: The Conclave — all five Eldrians

Images: eldrian-aethelin-[date].png, eldrian-velmara-[date].png

Reply: 'approve all' to schedule, or specific edits.

Quality Gates

Before saving to queue:

  • All 4 tweets per thread are under 280 chars
  • Each piece has mythological weight
  • No try-hard language, hashtags, or emojis (except Eldrian symbols)
  • Tweet 3 in each thread connects to present-day reality (AI/building/creation)
  • Images saved and paths confirmed

Output

Terminal display:

=== ARCANEA SWARM COMPLETE ===

Batch: arcanea-[date]
Pieces generated: [N]
Images: [list]
Saved to: data/arcanea-content-queue.json

Slack notification: sent to #content
