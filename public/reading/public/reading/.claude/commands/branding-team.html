/branding-team — FrankX Creative Branding Team

Orchestrates the full creative team for brand building, visual identity, and marketing materials.

Team Roster

Brand Strategist

  • Owns positioning, differentiation, and market narrative
  • Inputs: competitor analysis, user research, Frank identity
  • Outputs: positioning doc, brand pillars, voice spec

Visual Designer

  • Owns color, typography, imagery, component aesthetics
  • Inputs: brand pillars, design tokens, FRANK-Omega assets
  • Outputs: design system updates, visual direction, component designs

Copywriter

  • Owns all written brand touchpoints
  • Inputs: voice guidelines, product specs, audience context
  • Outputs: headlines, descriptions, CTAs, email copy

Character Director

  • Owns FRANK-Omega character library and placement
  • Inputs: page context, emotional tone, use case
  • Outputs: variant selection, placement specs, new variant briefs

Social Creative

  • Owns brand expression across LinkedIn, X, Instagram
  • Inputs: brand voice, character assets, content calendar
  • Outputs: post copy, visual direction, platform-specific adaptations

Workflow Stages

Stage 1 - Brief: What needs to be created? Who is it for? Stage 2 - Strategy: Brand Strategist sets direction Stage 3 - Create: Visual Designer and Copywriter work in parallel Stage 4 - Character: Character Director places FRANK-Omega correctly Stage 5 - Review: Does it feel like FrankX? Stage 6 - Adapt: Social Creative repurposes for platforms Stage 7 - Ship: Deploy with quality gates checked

FRANK-Omega Placement Guide

Primary (confident, hands on hips): Hero sections, main pages, default state Hero Flight (launching through data): Dynamic headers, action sections, launches City Chill (leaned back, MGK energy): Editorial, about sections, personality moments Portrait (close-up, blue eyes): Avatars, profile images, intimate CTAs Chibi Avatar (face portrait): Profile pictures, social icons, small formats Directing (pointing at viewer): CTAs, navigation callouts, directional moments Deep Work (analyzing, data orbiting): AI/research sections, processing states Chibi Tough (arms crossed): Social media, playful contexts, smaller formats Hey There (waving, walking): Onboarding, greet moments, welcome screens Cinematic Violet (full body, original): OG images, dramatic hero shots
