Swarm Council Update

Execute a full swarm intelligence update with all team councils reporting.

Arguments

  • $ARGUMENTS — Optional: specific team to focus on (dev, design, marketing, music, content, automation, all)

What this does

Activates the FrankX Intelligence Swarm system. Each council analyzes the current state and generates actionable plans:

Teams Available

  1. Dev Council — Code health, feature completion, tech debt, sprint plan
  2. Design Council — Visual consistency, accessibility, UX improvements
  3. Marketing Council — Content calendar, SEO priorities, growth experiments
  4. Music Empire — Track catalog, album progress, cover art pipeline
  5. Content Factory — Blog pipeline, distribution, book PDF status
  6. Automation Command — n8n workflow health, execution status, new automations
  7. Unified Command — Cross-team coordination, weekly execution map

Execution

If "$ARGUMENTS" is empty or "all":

Trigger the Weekly Intelligence Swarm:

curl -s -X POST "https://primary-production-ff336.up.railway.app/webhook/weekly-intelligence" \
  -H "Content-Type: application/json" \
  -d '{}'

For specific teams:

Analyze the codebase and current state for that team's domain, then post findings to #explorations (C09CQ1Y2TTK) via Slack MCP.

Always include:

  • Last week retrospective (commits, deployments, changes)
  • This week plan (daily priorities)
  • Proactive suggestions (3 actionable items)
  • Metrics dashboard (quantified progress)

Format with distinct team personalities and rich Slack mrkdwn formatting.
