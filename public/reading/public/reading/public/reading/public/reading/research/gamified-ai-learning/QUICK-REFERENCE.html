Gamified Learning Research - Quick Reference

One-Page Overview for Fast Decision Making

The Concept (30 seconds)

Transform your Claude Code skill.md files into a gamified learning system where:

  • Skills track XP, levels, and progress like an RPG
  • AI agents coach you with Socratic teaching and accountability
  • Content unlocks progressively as you demonstrate mastery
  • The Realm becomes a learning guild with mentors and challenges

Think: Duolingo + Habitica + Summit.AI, but embedded in Claude Code, creator-focused, and tied to real projects.

Why This Works (Research Validated)

✅ Streaks drive engagement - 7-day streaks = 3.6x retention (Duolingo data) ✅ AI tutoring effective - 50% time savings, 7% better outcomes (Khan Academy) ✅ Skill trees create clarity - Progressive revelation prevents overwhelm ✅ Accountability partners proven - Multiple successful products (Summit, Overlord, Rocky.ai) ✅ Gamification engages when done right - Must tie to real mastery, not just points

Core Mechanics

XP System:
├─ Read content: +5 XP
├─ Complete practice: +15 XP
├─ Pass mastery check: +25 XP
├─ Apply in real life: +40 XP
├─ Teach others: +50 XP
└─ 7-day streak bonus: +30 XP

Levels:
1→2: 100 XP    (Beginner)
2→3: 250 XP    (Developing)
3→4: 500 XP    (Competent)
4→5: 1000 XP   (Proficient)
5→10: ...      (Mastery path)

Unlocks:
Tier 1: Foundation (Free)
Tier 2: Intermediate (Level 3+)
Tier 3: Advanced (Level 6+)
Tier 4: Mentor (Level 9+)
