Agentic Creator OS

Personal AI Operating System for Creators

Turn your creative vision into reality with AI-powered multi-agent orchestration. ACOS helps you write, produce, launch, and grow—with agents that understand your voice and serve your mission.

Quick Start

# Initialize your creator hub
npx @frankx/agentic-creator-os init

# Start a multi-agent swarm
acos swarm start --task "Write a blog post about AI creativity"

# List available agents
acos agent list

# Browse the plugin marketplace
acos plugin marketplace

Commands

acos init

Initialize a new creator hub with interactive wizard.

acos init                    # Interactive 4-step wizard
acos init --yes              # Quick setup with defaults
acos init --name "Your Name" # Pre-fill your name

acos swarm

Multi-agent swarm orchestration.

acos swarm start --task "Write blog post"  # Start swarm for task
acos swarm status                          # Check swarm status
acos swarm checkpoint                      # Force alignment check
acos swarm stop                            # Stop the swarm

acos agent

Agent management.

acos agent list                    # List all agents
acos agent list --category Music   # Filter by category
acos agent spawn blog-writer       # Spawn specific agent
acos agent info blog-writer        # Get agent details

acos plugin

Plugin marketplace management.

acos plugin list                    # List all plugins
acos plugin marketplace             # Browse marketplace
acos plugin install acos-music      # Install plugin
acos plugin uninstall acos-music    # Uninstall plugin

acos serve

Run ACOS as MCP server for Claude Desktop, Cursor, or Windsurf.

acos serve                # Start MCP server
acos serve --port 3200    # Custom port

MCP Integration

Add ACOS to your Claude Desktop / Cursor / Windsurf configuration:

{
  "acos": {
    "command": "npx",
    "args": ["-y", "@frankx/agentic-creator-os", "serve"]
  }
}
