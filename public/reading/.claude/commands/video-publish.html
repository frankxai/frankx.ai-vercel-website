Video Publish

Purpose: Generate platform-specific metadata for a video, optimized for each platform's algorithm and format requirements, with a cross-promotion schedule.

Input

The user provides:

  1. Video title (required) — the title of the video to publish
  2. Video URL (optional) — YouTube link if already uploaded
  3. Source (optional) — production (from /video-produce output) or vault (from video-vault-100.json)

If source is production, read the matching entry from data/video-productions/. If source is vault, read from data/video-vault-100.json. Otherwise, generate metadata from the title alone.

Process

Step 1: Load Context

Read these files:

  • docs/strategy/CONTENT_VOICE_DNA.md — voice rules
  • data/video-annotations.json — check for existing clip annotations
  • data/clip-queue.json — check for queued clips from this video

If a production package exists in data/video-productions/, load the full script, tags, and description.

Step 2: Generate Platform Metadata

Generate optimized metadata for each platform:

Description: {full description with chapters, links, tags — max 5,000 chars}
Tags: {30 tags from production package or generated fresh}
Category: Science & Technology | Education | Entertainment
Thumbnail text: {2-4 words for overlay}
Publish time: {recommended — typically Tuesday/Thursday 9am EST}
End screen: {which video to link as "next"}
Cards: {timestamps where cards should appear}
Playlist: {which playlist to add to}

Step 7: Terminal Output

Display the complete publish package formatted for copy-paste. For each platform, output the ready-to-use text inside a code block so Frank can copy it directly.

=== PUBLISH PACKAGE: {Title} ===

Platforms: YouTube, TikTok, Instagram Reels, X/Twitter
Clips mapped: {N}
Schedule: 7-day cross-promotion

Character validation: {PASS/FAIL per platform}

Ready for copy-paste below.

Voice Rules (CRITICAL)

Same rules as /video-produce. Read docs/strategy/CONTENT_VOICE_DNA.md.

Adapt voice per platform:

  • YouTube: Full technical depth. No dumbing down.
  • TikTok: Punchy, visual, hook-first. Still no emojis.
  • Instagram: Slightly warmer but still data-backed. No guru energy.
  • X/Twitter: Sharpest version. Every word earns its place.

Across all platforms:

  • No "smash that like button"
  • No "follow for more"
  • No emojis
  • No "game-changer" / "revolutionary" / "unlock your potential"
  • Lead with results, specific numbers, or a contrarian take

Examples

Publish metadata for a new video:

/video-publish "Building multi-agent systems with ACOS v11"

Publish with existing YouTube URL:

/video-publish "ACOS v11 deep dive" https://youtube.com/watch?v=abc123

Publish from production package:

/video-publish "RAG pipeline tutorial" production
