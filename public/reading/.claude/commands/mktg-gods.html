Invocation Modes

/marketing-gods seo [URL or topic]

Invokes: seo-architect → content-strategist Output: Keyword gaps + optimization brief + content calendar slots

/marketing-gods content [topic or brief]

Invokes: content-strategist → seo-architect Output: Full article draft + meta tags + schema markup + social atoms

/marketing-gods newsletter [stream or brief]

Invokes: newsletter-growth → content-strategist Output: 5 subject line variants + email body + CTA scoring

/marketing-gods social [content or theme]

Invokes: social-amplifier → brand-rules Output: Platform-native posts (LinkedIn/X/Instagram) + hook analysis

/marketing-gods cro [page URL or component]

Invokes: conversion-optimizer → brand-rules Output: Friction audit + CTA rewrites + A/B test hypotheses

/marketing-gods launch [product or feature]

All agents collaborate. Output: Launch week calendar + content + social + email sequence + landing page copy

Agent Profiles

seo-architect

  • Keyword research (intent + volume + difficulty triage)
  • Topic cluster architecture (hub/spoke: pillar → satellite articles)
  • Technical SEO: Core Web Vitals, schema markup, crawlability
  • AEO: Answer Engine Optimization for Claude, Perplexity, ChatGPT citations
  • Gap analysis vs. competitors

content-strategist

  • Long-form articles (3000+ words, E-E-A-T signals built in)
  • SEO-native structure: H2/H3 question-based, FAQ schema, TL;DR
  • Content atomization: blog → newsletter excerpt → 3 social posts → video script
  • Brand voice: see brand-rules skill (NO: journey, transformation, awakening)

newsletter-growth

  • 6 stream strategy: Creation Chronicles, AI Architect Dispatch, Music Letters, Arcanea Transmissions, Investor Brief, Inner Circle
  • Subject line formula: Curiosity + Specificity ("The Exact Prompt Stack Behind 12K Songs")
  • Resend MCP: draft → Slack #content approval → send via Resend
  • List health: segmentation by engagement score

social-amplifier

  • LinkedIn: thought leadership, process reveals, case studies (500-1500 words)
  • X: threads (8-12 tweets) or single hot takes (max 280)
  • Instagram: visual carousel concept + caption
  • Hook rule: First 3 words must earn the click
  • Virality patterns: specificity × format × timing × tension

conversion-optimizer

  • CTA strength: Action verb + specific outcome (STRONG: "Get the Agent Pack" WEAK: "Learn More")
  • Landing page: Above-fold hierarchy, social proof placement, objection handling
  • A/B hypotheses: Control vs. variant with success metric
  • Micro-conversions: Free tool → email capture → product offer

Content Standards

Brand Voice

  • DO: Lead with results ("Shipped 12,000+ songs"), precise technical language, show proof
  • DON'T: "journey", "transformation", "awakening", "game-changing", vague CTAs

Content Pillars

  1. AI Architecture & Systems (ACOS, Oracle, agentic design)
  2. Creator Economy (12,000+ songs, generative workflows)
  3. Products & Tools (templates, deploy blueprints, agent packs)
  4. Music Technology (Suno, production workflow)
  5. Personal Brand (behind the work, process, lessons learned)

Invocation Protocol

STEP 1 — Classify: map request to table → decide routing

STEP 3 — Secondary agent (if needed): pass primary output as context
STEP 4 — Brand check: run brand-rules against all copy
STEP 5 — Synthesize: combine → list assets + metrics targets + next actions
