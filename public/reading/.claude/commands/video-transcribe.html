/video-transcribe — Video Transcription Pipeline

Purpose

Download and transcribe any YouTube video or local audio/video file into timestamped SRT subtitles, plain text transcript, and a structured summary.

Usage

/video-transcribe {url|file} [--language en] [--model medium]

Arguments

  • url|file — YouTube URL (https://youtube.com/watch?v=...) or local file path
  • --language — Language code (default: en). Auto-detect if omitted
  • --model — Whisper model size: tiny, base, small, medium (default), large-v3

Pipeline

Step 1: Source Resolution

  1. If YouTube URL:
    • Use yt-dlp to download audio only: yt-dlp -x --audio-format wav --audio-quality 0 -o "/tmp/transcribe/%(id)s.%(ext)s" "{url}"
    • Extract video metadata (title, duration, channel, upload date)
  2. If local file:
    • Verify file exists and is audio/video format
    • Extract metadata via ffprobe

Step 2: Transcription

Run faster-whisper (or whisper.cpp if installed):

# faster-whisper (Python)
faster-whisper-transcribe --model medium --language en --output_format all --output_dir /tmp/transcribe/ /tmp/transcribe/{id}.wav

# OR whisper.cpp (C++)
./main -m models/ggml-medium.bin -f /tmp/transcribe/{id}.wav -osrt -otxt -of /tmp/transcribe/{id}

Output files:

  • {id}.srt — timestamped subtitles
  • {id}.txt — plain text transcript
  • {id}.json — word-level timestamps (if available)

Step 3: Post-Processing

  1. Read the plain text transcript
  2. Generate a structured summary:
    • Title: Video title
    • Duration: Total length
    • Key Topics: 5-10 main topics discussed
    • Chapter Markers: Suggested timestamps for YouTube chapters
    • Quotes: Notable quotable moments with timestamps
    • Summary: 3-5 paragraph synopsis
  3. Save summary to data/video-transcripts/{id}-summary.md

Step 4: Integration

  1. Check if video exists in data/video-vault-100.json — if yes, add transcript reference
  2. Update data/video-annotations.json with chapter markers from transcript
  3. If --blog flag is set, trigger /video-to-blog with the transcript

Output Structure

data/video-transcripts/
├── {id}.srt          # Timestamped subtitles
├── {id}.txt          # Plain text
├── {id}-summary.md   # Structured summary
└── {id}-meta.json    # Metadata + processing info

Prerequisites

Check before running:

  • yt-dlp installed: pip install yt-dlp
  • faster-whisper installed: pip install faster-whisper (recommended)
  • OR whisper.cpp compiled with medium model downloaded
  • ffprobe available (comes with ffmpeg): apt install ffmpeg

If prerequisites are missing, provide install commands and exit gracefully.

Quality Checks

  • Verify SRT timestamps are sequential and non-overlapping
  • Verify transcript is non-empty and language matches expected
  • Log word count, duration, and words-per-minute ratio
  • Flag if WPM < 80 or > 200 (likely transcription quality issue)

Example

/video-transcribe https://youtube.com/watch?v=dQw4w9WgXcQ

→ Downloads audio, transcribes with medium model, generates SRT + summary, saves to data/video-transcripts/.
