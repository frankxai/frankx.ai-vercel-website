Video Clip

Purpose: Analyze a video from the vault, annotate timestamps with labels and types, score clip potential, and queue high-scoring clips for short-form extraction.

Input

The user provides ONE of:

  1. Video ID — a YouTube video ID from video-vault-100.json (e.g., VMj-3S1tku0)
  2. Video title — partial match against vault entries
  3. "batch" — process all unannotated vault entries

Optionally:

  • Transcript — if the user pastes a transcript, use it directly
  • Clip focus — e.g., "find the best quotes" or "extract tutorial segments"

Process

Step 1: Load Video Context

Read data/video-vault-100.json and find the matching entry. Extract:

  • Title, channel, duration, topic, tags
  • Any existing annotations from data/video-annotations.json

If the video has already been annotated, show existing annotations and ask whether to overwrite or append.

Step 2: Analyze Content

If a transcript is provided, analyze it directly. If not, use the video's metadata (title, tags, topic, channel) to generate intelligent timestamp suggestions based on:

Standard chapter structure by format:

For talks/lectures (30+ min):

  • Introduction / speaker context
  • Core thesis statement
  • Key examples or demonstrations
  • Q&A highlights
  • Closing takeaway

For tutorials (10-30 min):

  • Problem statement
  • Setup / prerequisites
  • Step-by-step implementation
  • Results / demo
  • Next steps

For interviews (45+ min):

  • Background / intro
  • Key question moments (3-5)
  • Controversial or surprising statements
  • Actionable advice
  • Closing thought

For short content (under 10 min):

  • Hook moment
  • Core insight
  • Best quote

Step 3: Generate Timestamp Annotations

For each identified moment, produce:

{
  "time": "MM:SS",
  "endTime": "MM:SS",
  "label": "Descriptive label for this moment",
  "type": "highlight | chapter | quote | clip-start | clip-end | key-insight",
  "viralScore": 7,
  "viralReason": "Why this segment has sharing potential",
  "platforms": ["youtube-shorts", "tiktok", "instagram-reels", "x-clip"]
}

Type definitions:

  • highlight — a standout moment worth bookmarking
  • chapter — a major section boundary (for YouTube chapters)
  • quote — a quotable statement, sharp and self-contained
  • clip-start / clip-end — boundaries of an extractable short-form clip
  • key-insight — a technical or strategic insight worth noting

Viral Score (1-10) criteria:

Score Criteria
9-10 Controversial take, surprising data, or "I need to share this" moment
7-8 Strong standalone insight, quotable, works without context
5-6 Good content but needs surrounding context to land
3-4 Useful reference but not shareable on its own
1-2 Transitional content, filler, or setup-only

Step 4: Identify Clip Candidates

From the annotations, identify extractable clips:

  • Duration: 15-60 seconds (shorts), 60-180 seconds (reels/extended)
  • Each clip must have a clear clip-start and clip-end pair
  • Each clip gets a suggested title for the short-form version
  • Each clip gets platform recommendations based on content type
=== CLIP CANDIDATES ===

  Start: {MM:SS} | End: {MM:SS} | Duration: {seconds}s
  Viral Score: {N}/10
  Reason: {Why this works as a standalone clip}
  Platforms: YouTube Shorts, TikTok, Instagram Reels
  Hook rewrite: "{First line if posted as a standalone short}"

Clip 2: ...

Step 5: Save Annotations

Save or update data/video-annotations.json:

{
  "videoId": "VMj-3S1tku0",
  "title": "Neural Networks: Zero to Hero",
  "channel": "Andrej Karpathy",
  "annotatedAt": "ISO date",
  "annotatedFrom": "metadata | transcript | manual",
  "chapters": [
    { "time": "0:00", "label": "Introduction" },
    { "time": "5:30", "label": "Building micrograd" }
  ],
  "annotations": [
    {
      "time": "12:45",
      "endTime": "13:30",
      "label": "Why backprop is all you need",
      "type": "quote",
      "viralScore": 8,
      "viralReason": "Contrarian take, self-contained, quotable",
      "platforms": ["youtube-shorts", "x-clip"]
    }
  ],
  "clips": [
    {
      "id": "clip-VMj-3S1tku0-1245",
      "startTime": "12:45",
      "endTime": "13:30",
      "duration": 45,
      "title": "Backprop is all you need",
      "viralScore": 8,
      "platforms": ["youtube-shorts", "tiktok"],
      "hookRewrite": "Every complex neural network training trick reduces to one thing.",
      "status": "queued"
    }
  ]
}

If data/video-annotations.json already exists, merge the new entry (keyed by videoId). Do not overwrite other video entries.

Step 6: Queue High-Scoring Clips

Any clip with viralScore >= 7 gets added to data/clip-queue.json:

{
  "id": "clip-{videoId}-{startTimeCompact}",
  "videoId": "VMj-3S1tku0",
  "videoTitle": "Neural Networks: Zero to Hero",
  "channel": "Andrej Karpathy",
  "clipTitle": "Backprop is all you need",
  "startTime": "12:45",
  "endTime": "13:30",
  "duration": 45,
  "viralScore": 8,
  "platforms": ["youtube-shorts", "tiktok"],
  "hookRewrite": "Every complex neural network training trick reduces to one thing.",
  "status": "queued",
  "queuedAt": "ISO date",
  "extractedAt": null,
  "publishedAt": null
}

Status lifecycle: queued -> extracted -> reviewed -> published

Display a summary:

Clips: {N} candidates ({M} queued with viral score >= 7)

  1. "{title}" — {duration}s — Score: {N}/10
  2. "{title}" — {duration}s — Score: {N}/10

Files updated:
  - data/video-annotations.json ({videoId} entry)
  - data/clip-queue.json (+{M} clips)

Batch Mode

When invoked with batch:

  1. Read all entries from data/video-vault-100.json
  2. Read existing annotations from data/video-annotations.json
  3. Process only videos without existing annotations
  4. Generate metadata-based annotations for each
  5. Report totals at the end

Examples

Annotate a specific video:

/video-clip VMj-3S1tku0

Annotate with a pasted transcript:

/video-clip dWr1eTeK6p4
[paste transcript below]

Annotate with clip focus:

/video-clip 5MUvJv_sB64 "find the best quotes about AGI timeline"

Batch annotate all unprocessed vault entries:

/video-clip batch

Process the annotation now.
