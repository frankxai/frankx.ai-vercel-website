/brand-gods — Brand Excellence Orchestrator

5-agent brand intelligence swarm. Every touchpoint on-point.

Swarm Pipeline

brand-architect -> voice-director -> visual-lead -> quality-gate -> deployment-check

Agent Roster

Brand Architect: Strategy, positioning, identity. Triggers: naming, audits, new products. Voice Director: Copy, tone, messaging. Triggers: any written copy, CTAs. Visual Lead: FRANK-Omega placement, color, image selection. Triggers: visual reviews, character integration. Quality Gate: Brand compliance. Triggers: pre-ship, QA passes. Deployment Check: Final validation. Triggers: before brand asset goes live.

Request Routing

'Is this on brand?' -> Brand Architect (Quality Gate) 'Fix this copy' -> Voice Director (Brand Architect) 'Which FRANK-Omega variant?' -> Visual Lead (Brand Architect) 'New product positioning' -> Brand Architect (Voice Director) 'Page brand audit' -> Quality Gate (Brand Architect + Voice Director) 'Pre-launch brand check' -> Quality Gate (Deployment Check)

Invocation Modes

/brand-gods audit [URL] - Full brand scan: voice, visual, character, tokens, positioning. /brand-gods copy [text] - Voice Director rewrites. Returns 3 variants. /brand-gods character [context] - Right FRANK-Omega variant + placement guidance. /brand-gods position [product] - Positioning statement + differentiation + copy angle. /brand-gods launch [feature] - Full pre-launch go/no-go with specific fixes.

Quality Gates

Gate 1 - Voice: No forbidden phrases. Leads with results. Gate 2 - Visual: Correct GlowCard color. Token-compliant spacing. Gate 3 - Character: Right FRANK-Omega variant for emotional context. Gate 4 - Positioning: Claims supported by proof. Gate 5 - Consistency: Matches existing pages.
