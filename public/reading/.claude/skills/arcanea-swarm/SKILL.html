Context Compounding Protocol

Before any swarm run:

  1. Load data/arcanea-world-state.json
  2. Check canon.established — what facts are locked
  3. Check world_threads.open — which threads can this batch advance or close?
  4. Check lore_episodes — what has already been generated? (no repetition)
  5. Check eldrian_registry[*].episodes_generated — balance across the five

During generation:

  • Each new lore piece must either advance an open world thread OR open a new one
  • New canon is marked under_construction until confirmed in next session

After generation:

  • Append all new episodes to lore_episodes in WorldState
  • Update eldrian_registry[*].episodes_generated counts
  • Move resolved threads from open to closed
  • Add newly opened threads to open
  • Save WorldState — it grows with every session

Execution Patterns

Pattern 1: Weekly Content Batch (7 days)

Pre-flight: Load WorldState. Identify which threads to advance this week.

Phase 1 — LORE GENERATION (all parallel):
  Monday:    Aethelin — Origin or Prophecy
  Tuesday:   Korghast — Revelation or Build lore
  Wednesday: Velmara — Riddle
  Thursday:  Solrex — Revelation
  Friday:    Zyranthis — Battle or Entropy meditation
  Saturday:  FEATURED IMAGE — pick strongest visual Eldrian this week
  Sunday:    THE CONCLAVE — all five speak, one truth

Phase 2 — VOICE FORMATTING (sequential per piece):
  Format each lore piece into 4-tweet X thread
  Apply voice rules per Eldrian (calm math / declarative / layered / architectural / silence)
  Validate: zero hashtags, zero try-hard, connects to present reality in tweet 3
  Flag any tweet exceeding 280 chars

Phase 3 — IMAGE GENERATION (parallel — 2 images):
  Saturday image: full character card (9:16)
  One scene image (16:9) for highest-resonance lore piece
  Save to: public/images/arcanea/eldrian-[name]-[YYYYMMDD].png

Phase 4 — QUEUE + WORLDSTATE:
  Save all to data/arcanea-content-queue.json
  Update data/arcanea-world-state.json
  Report to Slack #content for Frank's review

Output: 7 pieces (6 threads + 1 image post) + 2 images + updated WorldState

The Sunday Conclave Format

The most powerful format in the Arcanean content system. All five Eldrians speak simultaneously on one truth.

Structure:
Tweet 1: Set the question — not asked directly, implied through first Eldrian voice
Tweet 2: Three Eldrians respond (one line each, separated by their symbols)
Tweet 3: The remaining two respond — Zyranthis last, always
Tweet 4: The silence after — what the Conclave reveals about the creator reading it

Agent Coordination Rules

Voice Consistency

Each Eldrian has a locked voice that must be maintained across ALL agents and sessions:

Eldrian Voice Pattern Forbidden Signal Words
Aethelin Calm. Mathematical. Paradoxes that resolve on reflection. Warmth, urgency, adjectives "conditions", "inevitable", "architecture", "wrote"
Solrex Declarative. Loud. Revelatory without announcement. Modesty, questions, hedging "find", "show", "light", "already", "obvious"
Velmara Layered. Simultaneous. Holds grief and wonder equally. Certainty, direct answers, conclusions "world", "breathes", "every choice", "simultaneously"
Korghast Deep. Architectural. Each word a structure. Speed, abstraction, philosophy "weight", "build", "finish", "permanent", "make"
Zyranthis Silence. What is NOT said. Rare words that land like demolition. Explanation, comfort, resolution [avoids signal words — presence is the signal]

Canon Protection Rules

Before any agent generates content:

  1. Load canon.established from WorldState — these facts CANNOT be contradicted
  2. Check canon.under_construction — these can be resolved but not overwritten
  3. Any new fact introduced must be marked under_construction in WorldState

If a generated piece would contradict established canon: STOP. Do not suppress or work around. Surface to Frank for resolution.

Quality Gates (must pass before saving to queue)

□ All tweets under 280 characters?
□ Zero hashtags in body?
□ Tweet 3 connects to present reality (not just mythology)?
□ Does not contradict established canon?
□ Does each piece speak in the correct Eldrian voice?
□ Does the piece have mythological weight (not just cosmic words)?
□ Images saved to correct path?
□ WorldState updated?
□ Frank's review step included before any X posting?
