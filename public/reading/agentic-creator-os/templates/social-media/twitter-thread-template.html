Platform-Specific Formatting

Character Limits

  • Each tweet: 280 characters max
  • Optimal length: 200-250 characters (leaves room for retweets with comments)

Visual Elements

  • Emojis: Use 1-2 per tweet for visual breaks (🧵 ↓ ✨ 🔥 💡 ⚡)
  • Line breaks: Use blank lines to create visual breathing room
  • Numbers: Use for lists (1. 2. 3.) or bullet points (• or -)
  • Bold/Italics: Not supported, use CAPS or "quotes" for emphasis

Hashtags

  • Avoid: Hashtags reduce engagement on Twitter
  • Exception: Branded hashtags or community-specific tags

Mentions

  • First tweet: Don't mention anyone (reduces reach)
  • Later tweets: Mention relevant accounts for engagement
  • Last tweet: Mention yourself to encourage follows

Pro Tips

Timing

  • Post when your audience is most active (check analytics)
  • Pin the thread to your profile for 24-48 hours
  • Repost the thread 1-2 weeks later with a new hook

Engagement Hacks

  • Reply to your own thread with additional insights
  • Ask a question in tweet 5-7 to boost mid-thread engagement
  • Quote tweet your own thread with a key takeaway

Thread Length

  • Optimal: 8-12 tweets
  • Minimum: 5 tweets (shorter = not worth threading)
  • Maximum: 15 tweets (longer = completion rate drops)

Testing Hooks

  • Write 5 different hooks for the same thread
  • Test them as standalone tweets first
  • Use the highest-performing hook for your thread

Customization Checklist

Before posting, verify:

  • Hook creates curiosity or controversy
  • Each tweet provides standalone value
  • Thread flows logically from point to point
  • Personal story or proof included
  • Clear call to action at the end
  • No typos or formatting errors
  • Emojis used sparingly for visual breaks
  • Thread length is 8-12 tweets
  • First tweet doesn't mention anyone
  • Last tweet includes follow CTA

Common Mistakes to Avoid

❌ Starting with "Thread:" or "1/" (wastes hook space) ❌ Making every tweet the same length (creates visual monotony) ❌ Saving the best insight for last (front-load value) ❌ Using too many hashtags (reduces reach) ❌ Not engaging with replies (kills algorithmic boost) ❌ Posting without a clear CTA (missed conversion opportunity) ❌ Writing for yourself instead of your audience (ego vs. value)

✅ Hook with curiosity or controversy ✅ Vary tweet length for visual rhythm ✅ Deliver value in every single tweet ✅ Skip hashtags, focus on quality content ✅ Reply to every comment in first hour ✅ Clear CTA that serves your goals ✅ Write for your reader's transformation
