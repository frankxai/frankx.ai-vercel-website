# AGENTS.md

This project's agent rules live in [`CLAUDE.md`](./CLAUDE.md). They are tool-agnostic — they
apply to Cursor, Copilot, Codex, Gemini, and any other coding agent just as much as to Claude
Code.

If your harness reads `AGENTS.md` but not `CLAUDE.md`, copy the contents of `CLAUDE.md` here,
or symlink them. Keep one source of truth so the rules can't drift apart.
