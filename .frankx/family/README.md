# Private family records

Real family records are not stored in this public repository or its generated reading mirror.

The Family Intelligence System keeps living-person data, family relationships, guardianship,
continuity instructions, and private evidence in the authenticated family tenant. Public source
may contain protocols, synthetic examples, and redacted material only.

See `/familie` for the protected German portal shell and
`/familien-intelligenz-system` for the public architecture overview.
