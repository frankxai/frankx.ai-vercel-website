Email + PDF Delivery System (Resend + Vercel Blob)

Project: frankxai
Branch: feature/email-pdf-delivery

**Feature**: Resend Email + Vercel Blob PDF Delivery
**Purpose**: Automate email capture with instant PDF delivery

**Step 1: Upload PDF to Vercel Blob (One-time)**
```bash
# Install Vercel CLI
npm i @vercel/blob

# Upload PDF to Vercel Blob Storage
# Go to Vercel Dashboard ‚Üí Your Project ‚Üí Storage ‚Üí Create Blob Store
# Upload: /pdfs/ai-agent-starter-kit.pdf
# Get URL: https://[random].public.blob.vercel-storage.com/starter-kit.pdf
```

**Step 2: Install Dependencies**
```bash
npm install resend @vercel/postgres
```

**Step 3: Environment Variables**
```
RESEND_API_KEY=re_xxxx
BLOB_READ_WRITE_TOKEN=vercel_blob_xxx
POSTGRES_URL=postgres://xxx
PDF_BLOB_URL=https://[your-blob-url]/starter-kit.pdf
```

**Step 4: Database Schema (Vercel Postgres)**
```sql
CREATE TABLE subscribers (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  email TEXT NOT NULL UNIQUE,
  lead_magnet TEXT,
  subscribed_at TIMESTAMP DEFAULT NOW()
);
```

**Step 5: API Route** `/app/api/subscribe/route.ts`
```typescript
import { sql } from '@vercel/postgres';
import { Resend } from 'resend';
import { NextResponse } from 'next/server';

const resend = new Resend(process.env.RESEND_API_KEY);

export async function POST(request: Request) {
  try {
    const { email } = await request.json();

    // Validate email
    if (!email || !email.includes('@')) {
      return NextResponse.json({ error: 'Invalid email' }, { status: 400 });
    }

    // Save to database
    await sql`
      INSERT INTO subscribers (email, lead_magnet)
      VALUES (${email}, 'ai-agent-starter-kit')
      ON CONFLICT (email) DO NOTHING
    `;

    // Send email with PDF attachment
    await resend.emails.send({
      from: 'frank@frankx.ai',
      to: email,
      subject: 'Your AI Agent Starter Kit üöÄ',
      html: `
        
Thanks for subscribing!

        
Your AI Agent Starter Kit is attached to this email.

        
Get ready to build your agent system!

        
- Frank

      `,
      attachments: [{
        filename: 'ai-agent-starter-kit.pdf',
        path: process.env.PDF_BLOB_URL
      }]
    });

    return NextResponse.json({ success: true });
  } catch (error) {
    console.error('Subscribe error:', error);
    return NextResponse.json({ error: 'Failed to subscribe' }, { status: 500 });
  }
}
```
