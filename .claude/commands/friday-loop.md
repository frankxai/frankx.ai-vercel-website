2. Pull performance on everything `Published` this week + trailing 28d movers.
3. Higgsfield-analyze own top and bottom performer: what the hook/structure actually did.

5. Notion: items → `Analyzed` with metric snapshot; update pillar scoreboard.
6. Seed next week: adjust Monday sweep queries based on what the data said; write the handover brief into Ops Log (next session reads this — no re-explaining).
7. End: 5-line verdict to Frank — what worked, what dies, what doubles.
