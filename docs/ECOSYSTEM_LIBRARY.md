## ACOS - Agentic Creator OS

### What is ACOS?

ACOS (Agentic Creator OS) is a **Claude Code-native productivity system** that transforms how creators work with AI.

**Philosophy:** Configuration-first, zero-install, Claude-native

### Core Components

#### 1. Commands (25)

| Category | Commands | Purpose |
|----------|----------|---------|
| **Creation** | `/article-creator`, `/create-music`, `/infogenius`, `/factory` | Content production |
| **Strategy** | `/starlight-architect`, `/council`, `/research`, `/plan-week` | Planning & decisions |
| **Development** | `/spec`, `/nextjs-deploy`, `/ux-design`, `/automation-dev` | Building & shipping |
| **System** | `/acos`, `/planning-with-files`, `/inventory-status`, `/mcp-status` | Management |
| **Quality** | `/review-content`, `/classify-content`, `/polish-content` | Excellence |

#### 2. Smart Router (`/acos`)

Single entry point that auto-routes to appropriate command:

```
User: "write a blog post about AI agents"
           │
           ▼
┌─────────────────────┐
│    /acos Router     │
│  Parse: "blog post" │
│  Route: /article-   │
│         creator     │
└─────────────────────┘
           │
           ▼
┌─────────────────────┐
│  /article-creator   │
│  + content-creation │
│    skill profile    │
└─────────────────────┘
```

#### 3. Skill Profiles (5 Optimized Sets)

| Profile | Skills | Triggers |
|---------|--------|----------|
| `frankx-web` | Next.js, React, Vercel, shadcn | `app/**/*.tsx` files |
| `content-creation` | SEO, brand, copywriting | `content/**/*.mdx` |
| `music-production` | Suno AI mastery | "suno", "music" keywords |
| `oracle-enterprise` | OCI, Oracle ADK | "oracle" keywords |
| `agent-development` | MCP, agents, langgraph | `.claude/**/*` files |

#### 4. Agents (6 Core + 4 Prompt Library)

**Core Agents:**
- Technical Architect - AI systems design
- Music Producer - Suno AI creation
- Content Engine - Content & products
- SEO Intelligence - Search optimization
- Frontend Designer - UI/UX development
- DevOps Engineer - Deployment & CI/CD

**Prompt Library Agents:**
- Prompt Harvester - Pattern research
- Prompt Architect - Advanced prompt design
- Prompt Tester - Quality validation
- Prompt Evolver - Continuous improvement

## Process Documentation

### Process 1: Skill Activation

```
TRIGGER DETECTION
       │
       ├── File Pattern Match?
       │   └── YES → Load profile for matching pattern
       │
       ├── Keyword Detected?
       │   └── YES → Load skills with keyword trigger
       │
       ├── Command Invoked?
       │   └── YES → Load command-mapped skills
       │
       └── No Match?
           └── Load core skills only

SKILL LOADING
       │
       ├── Check profile priority order
       ├── Load max 5 concurrent skills
       ├── Exclude blocked patterns
       └── Merge with always-available core skills

EXECUTION
       │
       ├── Skills provide context to agent
       ├── Agent executes with skill knowledge
       └── Track activation in analytics
```
