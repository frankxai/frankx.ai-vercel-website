# Strategy Directory

This directory keeps the FrankX.ai strategic canon explicit and versioned.

- `v1/` captures the historical creator + enterprise mix so we can reference past positioning without keeping it in active circulation.
- `v2/` contains the current Creator-First blueprint, daily operating rules, and agent instructions that every model must follow going forward.
- `v2/FRANKX_BRAND_CONTENT_MEDIA_OS_2026.md` is the private strategic source for FrankX brand, content, media, domain, and site posture.
- `v2/FRANKX_CONTENT_COMMAND_CENTER_2026.md` is the private execution doctrine for content, social, SEO, ethical influence, and insight capture.
- `v2/FRANKX_FOUNDER_LED_MEDIA_SWARM_2026.md` is the private weekly media-swarm operating manual for founder-led publishing.
- `v2/FRANKX_SIGNAL_LOOP_90_DAY_PLAN.md` is the first 90-day newsletter/social/media calendar.
- `v2/frankx-insight-card.schema.json` is the private insight card schema for capture, gating, and learning.
- `v2/frankx-publish-board.schema.json` is the private publish-board schema for planning, gating, repurposing, and learning.
- `v2/frankx-social-packet.schema.json` is the private weekly social-packet schema for source-traced platform assets.

Move any deprecated plans, roadmaps, or audience experiments into `v1/` and author future-facing guidance in `v2/`. Each file should open with purpose, owner, and last-updated date so agents know how fresh the material is.
