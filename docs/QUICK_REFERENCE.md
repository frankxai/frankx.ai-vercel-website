## üìö Writing Workflows

### Blog Post Workflow
1. "Research [topic] for article"
2. "Create outline for [topic]"
3. "Write first draft"
4. "Add SEO: TL;DR, FAQ, meta"
5. "Review and polish"
6. Deploy with `sync-to-production.sh`

### Technical Tutorial Workflow
1. "Outline tutorial for [topic]"
2. "Write step-by-step guide"
3. "Add code examples"
4. "Create diagrams if needed"
5. "Review for accuracy"

### Book Chapter Workflow
1. "Outline chapter: [title]"
2. "Write section: [section]"
3. "Add examples and stories"
4. "Review and refine"
5. "Check for continuity"
