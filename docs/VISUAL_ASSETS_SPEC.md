## Infographic Specifications

### VIS-001: ACOS Architecture

**Prompt:**
```
Create a 16:9 3D isometric infographic showing the ACOS (Agentic Creator OS) architecture.

VISUAL STYLE: 3D Isometric Render. Claymorphism or high-gloss plastic texture, studio lighting, soft shadows, looks like a physical model.

AUDIENCE: Target: Industry Professional. Clean, modern tech aesthetic.

INCLUDE:
- Central hub labeled "ACOS Smart Router (/acos)"
- 5 command category blocks: Creation (8), Strategy (5), Development (4), System (5), Quality (3)
- 5 skill profile modules floating: Web, Content, Music, Oracle, Agent Dev
- 6 agent avatars around the perimeter
- Data flow arrows connecting components
- Badge showing "630+ Skills" and "25 Commands"

COMPOSITION:
- Isometric 3D view from 45-degree angle
- Color scheme: Deep blues (#1a1f2e), tech cyan (#00d4ff), purple accents (#8b5cf6)
- White background with subtle grid
- Text labels should be large and legible
- FrankX logo watermark bottom-right
```

Create a 16:9 futuristic cyberpunk HUD infographic showing the Arcanea 10 Gates progression system.

VISUAL STYLE: Cyberpunk HUD. Glowing neon blue/cyan lines on dark background, holographic data visualization, 3D wireframes.

AUDIENCE: Target: Creative Professional. Mystical yet technological feel.

COMPOSITION:
- Dark void background (#0a0f1c)
- Neon cyan primary (#00d4ff)
- Purple secondary (#8b5cf6)
- Gold accents (#f59e0b) for achievements
- Holographic grid overlay
- Ethereal glow effects on active elements
```

Create a 16:9 technical blueprint infographic showing the FrankX Unified Architecture with 6 layers.

VISUAL STYLE: Da Vinci Notebook style. Ink on parchment sketch, handwritten annotation style, rough but accurate lines, technical precision.

AUDIENCE: Target: System Architect. Detailed technical schematic.

### VIS-005: Skill Activation Process

**Prompt:**
```
Create a 16:9 Bauhaus minimalist infographic showing the skill activation decision tree.

VISUAL STYLE: Bauhaus Minimalist. Flat vector art, limited color palette (3 colors), reliance on negative space and simple geometric shapes.

AUDIENCE: Target: Developer. Technical process flow.

INCLUDE:
- Decision diamond: "Trigger Detected?"
- Three paths: File Pattern, Keyword, Command
- Each path leads to: "Load Profile"
- Profile shows: "5-10 Skills Activated"
- Final step: "Execute with Agent"
- Show excluded patterns being filtered out

COMPOSITION:
- Pure white background
- Three colors only: Black, Cyan (#00d4ff), Purple (#8b5cf6)
- Geometric shapes (circles, squares, triangles)
- Heavy use of negative space
- Sans-serif typography
- No gradients, no shadows
```

### VIS-008: Content Publishing Pipeline

**Prompt:**
```
Create a 16:9 flow diagram infographic showing the FrankX content publishing pipeline.

VISUAL STYLE: Modern flow diagram. Clean lines, rounded nodes, gradient connections.

AUDIENCE: Target: Content Manager. End-to-end publishing flow.

INCLUDE:
- Flow from left to right:

COMPOSITION:
- Light gradient background
- Rounded rectangle nodes
- Gradient connector lines
- Icon + label for each step
- Color coding: private (purple), public (cyan), storage (orange)

## Style Guide for All Visuals

### Brand Colors
```css
--void-black: #0a0f1c;
--space-blue: #1a1f2e;
--tech-cyan: #00d4ff;
--premium-purple: #8b5cf6;
--gold-accent: #f59e0b;
--clean-white: #ffffff;
```

### Typography
- Headlines: Bold sans-serif (Inter, Geist)
- Labels: Medium weight
- Annotations: Regular weight

### Consistency Requirements
- All infographics 1920x1080 (16:9)
- FrankX watermark on all images
- Consistent icon style across set
- Accessible color contrast
