# OpenAI DevDay Resource Spec

Date: 2026-07-03

## Objective

Create a production-ready FrankX resource funnel that turns official OpenAI DevDay learning into a practical builder path, a free workbook lead magnet, and a bridge into the GenCreator community.

## Audience

- AI architects comparing OpenAI surfaces.
- Creators who want repeatable agent workflows without becoming full-time platform engineers.
- Operators and founders who need a current OpenAI map before committing to tools.

## Routes

- `/learn/openai-devday-agent-stack`
- `/learn/codex-mastery`
- `/learn/chatgpt-mastery`
- `/guides/openai-devday-agent-stack-2026`
- `/resources/openai-agent-builder-workbook`

## Design Direction

Keep the surface quiet, technical, and utility-first. Use emerald/cyan FrankX cues with restrained dark UI, real source links, and clear artifact hierarchy. Avoid fake product screenshots, oversized marketing claims, and abstract decoration that does not help the user decide what to do next.

## Asset Tier

Tier C for this change: exact UI/system/content assets are the correct medium. The primary artifact is the workbook funnel and public Markdown pack, not a generated hero image.

## SEO / AEO Intent

Answer these queries directly:

- What should I watch from OpenAI DevDay?
- What is the current OpenAI agent stack?
- Should I use Agent Builder in 2026?
- How do I choose between Codex, Agents SDK, Apps SDK, Workspace Agents, and ChatGPT agent mode?
- Is there a free OpenAI agent workbook?

## Quality Gates

- TypeScript and content checks pass.
- Static internal link check passes or any unrelated existing failures are documented.
- Visual QA covers desktop and mobile for the new routes.
- Design evidence validates before final handoff.
- Production deployment verified on the live FrankX URL.
